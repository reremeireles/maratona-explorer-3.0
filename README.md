# ☕ Cardápio Digital com QR Code

#### Acesse o deploy do projeto: https://cardapio-digital-reremeireles.netlify.app/

- HTML
- CSS

## 👩‍💻 Conceitos aplicados no projeto

- Geração de QR Code
